# SermonPod
A macOS+Linux tool for extracting audio from YouTube videos, enabling easy editing and multi-platform distribution of sermon content for podcasting.

## Overview
SermonPod is designed to streamline the workflow of converting YouTube sermon videos into podcast-ready audio files. The tool extracts MP3 files from YouTube videos, which can then be edited in your preferred audio editing software and uploaded to multiple podcast platforms.

## Requirements
- macOS or Ubuntu-based Linux distribution

## Usage
1. Make the program executable. You can do this via terminal command for now. 
```bash
chmod +x sermonpod.sh
```
2. Run the script in the terminal or double click it in your File Browser
```bash
./sermonpod.sh
```
